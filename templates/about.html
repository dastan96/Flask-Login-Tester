Flask Login Tester is a Flask-based web application designed to:

  • Provide secure user authentication.
  • Automate API testing using pytest.
  • Store test results dynamically in an SQLite database.
  • Display real-time test results on a visually appealing dashboard.
  • Automate test execution using GitHub Actions every Sunday at midnight.
